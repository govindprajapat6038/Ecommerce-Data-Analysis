# E-Commerce Data Analysis

## Project Overview
This project analyzes an e-commerce retail dataset to uncover sales trends, customer behavior, and top-performing products using Python.

## Dataset
- Source: Kaggle Online Retail Dataset
- Records: 541,909

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Data Cleaning
- Revenue Calculation
- Top Selling Products
- Top Revenue Products
- Country-wise Revenue
- Monthly Sales Trend
- Data Visualization
